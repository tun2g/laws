# Available Skill Resources

Curated registry of external skill collections for discovering existing skills before creating new ones.
MUST download the entry point to know what's available — do not rely on secondhand summaries or lists, it's just some highlights and may be outdated.

**IMPORTANT**: Entry point content is large, prefer downloading it to /tmp, if you use WebFetch, MUST provide the prompt to search for your target domain in the raw content and return with real according URLs to follow up — do not summarize as it will lead to loss of information.

## Skill Repositories

### VoltAgent/awesome-agent-skills (Recommended)

- **Repo**: https://github.com/VoltAgent/awesome-agent-skills
- **Entry point**: https://raw.githubusercontent.com/VoltAgent/awesome-agent-skills/main/README.md
- **What**: 380+ curated agent skills from official dev teams (Anthropic, Vercel, Cloudflare, Stripe, Google, Microsoft, Hugging Face, Expo, Sentry, and more) plus community contributions. Compatible with Claude Code, Codex, Gemini CLI, Cursor, Copilot, and others. Focuses on real-world skills, not mass-generated content.

### hesreallyhim/awesome-claude-code

- **Repo**: https://github.com/hesreallyhim/awesome-claude-code
- **Entry point**: https://raw.githubusercontent.com/hesreallyhim/awesome-claude-code/main/README.md
- **What**: Curated index of hundreds of Claude Code resources organized by type — agent skills, workflows & knowledge guides, tooling (IDE integrations, orchestrators, config managers), status lines, hooks, slash-commands, CLAUDE.md files, and alternative clients.

### ComposioHQ/awesome-claude-skills

- **Repo**: https://github.com/ComposioHQ/awesome-claude-skills
- **Entry point**: https://raw.githubusercontent.com/ComposioHQ/awesome-claude-skills/main/README.md
- **What**: 33+ practical Claude Skills across document processing, development tools, data analysis, business/marketing, creative/media, security, and productivity. Includes Composio automation skills for 78+ SaaS apps (CRM, project management, communication, storage, social media).
